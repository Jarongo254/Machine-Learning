**Multivariate data?**
- No. of variables, how to explore them, which ones are important
- what are the dimensions, can they be reduced, visualized (how- PCA, MDS, ICA, SNHA)
- Can they be grouped? based on what? approaches - clustering, neural networks, SNHA

These approaches fall under **unsupervised learning**:
* The classes for the data points/variables is largely unknown and can only be inferred based on their prevailing features (commonalities and differences) such as distance from each other(euclidian, manhattan...)

This differs from **supervised learning** where data features of available labeled training data are used as a function to predict features of unknown data
